# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
1. Install NodeJs 16.x and NPM on Ubuntu
2. Update the backend URL in the src/TodoApp.js 
3. Run npm install
4. RUn npm run build
5. Copy the Generated artifacts in nginx server
6. Container images are built and stored in Azure Container Registry (ACR).
7. Application deployments are managed through ArgoCD using a GitOps workflow.
8. Applications are deployed to Azure Kubernetes Service (AKS) and exposed through LoadBalancer services.
